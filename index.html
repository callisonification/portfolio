The 3 phases of Web Development:

Phase 1 : Design

All applications start with a design, a vision of the final solution. These visual representations allow you to see the final product before any code has been written. Allowing for quick changes and multiple versions.

Phase 2 : Code

After the design phase the actual site is built using the latest web technologies. All applications are built with user experience in mind according to today’s best practices and design methodologies.

Phase 3 : Launch

Once the application is built, has been tested across all browsers and devices, and is completely bug and glitch free it is ready for launch. I can assist with installing and even hosting the application for you.
